# 🤝 GchBBS-JP 貢献ガイドライン

GchBBS-JPへようこそ！あなたの貢献を心より歓迎します。本プロジェクトは、2ch/5ch文化を継承する**次世代BBSソフトウェアをGo言語でゼロから構築**することを目標としています。

## 1. 議論の場について
* **仕様/企画の議論:** すべて **GitHub Discussions** のトピックで行います。
* **タスク/バグ報告:** **GitHub Issues** を使用してください。

## 2. 行動規範と技術スタック
* **行動規範:** すべての参加者は、技術的な議論に集中し、敬意と思いやりをもってコミュニケーションを行ってください。
* **技術スタック:** Core: **Go (Golang)**, DB: **PostgreSQL**, License: **MIT License**

## 3. 貢献の進め方
1.  **Issue (タスク) の選択:** Issuesリストから、未アサインのタスクを選択し、コメントで「担当します」と宣言してください。
2.  **ブランチの作成:** 常に `main` ブランチからフィーチャーブランチ（例: `feature/issue-1-aa-css`）を作成してください。
3.  **プルリクエスト (PR) の提出:** 以下の情報を含めてください。
    * **関連Issue:** どのIssueを解決したか（例: `Fixes #1`）
    * **変更内容:** 簡潔に記述してください。

## 4. 開発環境
Docker (Issue #1) を用いた環境構築を推奨します。詳細はIssue #1を参照してください。
